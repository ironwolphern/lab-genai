# 🧠 Lab GenAI - Laboratorio de IA Generativa con Python

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![AI](https://img.shields.io/badge/AI-Generative-purple.svg)](https://github.com/topics/artificial-intelligence)

Un laboratorio completo de IA generativa con Python que incluye implementaciones y documentación de los principales frameworks para el desarrollo de sistemas multiagente y aplicaciones de IA.

## 📋 Descripción

Este repositorio contiene ejemplos prácticos, tutoriales y documentación para aprender y experimentar con diferentes frameworks de IA generativa. Cada rama se enfoca en una tecnología específica, desde conceptos básicos de Python hasta frameworks avanzados de sistemas multiagente.

## 🌟 Características

- 📚 Documentación completa de cada framework
- 🔧 Ejemplos prácticos y funcionales
- 🚀 Implementaciones paso a paso
- 📖 Manuales de aprendizaje progresivo
- 🤖 Sistemas multiagente avanzados

## 🌿 Ramas del Repositorio

### 🐍 [main](https://github.com/tu-usuario/lab-genai/tree/main) - Python Básico
**Rama principal con fundamentos de Python**
- Conceptos básicos de Python
- Estructuras de datos y algoritmos
- Programación orientada a objetos
- Manejo de archivos y APIs
- Preparación para IA y machine learning

### 🤖 [autogen](https://github.com/tu-usuario/lab-genai/tree/autogen) - Microsoft AutoGen
**Framework para sistemas multiagente conversacionales**
- Configuración de agentes conversacionales
- Sistemas multiagente colaborativos
- Automatización de tareas complejas
- Integración con LLMs
- Casos de uso empresariales

### 🛠️ [crewai](https://github.com/tu-usuario/lab-genai/tree/crewai) - CrewAI
**Framework para equipos de agentes IA especializados**
- Creación de equipos de agentes especializados
- Coordinación de tareas entre agentes
- Roles y responsabilidades definidas
- Workflows automatizados
- Integración con herramientas externas

### 📊 [langGraph](https://github.com/tu-usuario/lab-genai/tree/langGraph) - LangGraph
**Framework para aplicaciones stateful con LLMs**
- Grafos de estado para aplicaciones IA
- Workflows complejos con LangChain
- Manejo de estado y memoria
- Aplicaciones conversacionales avanzadas
- Integración con bases de datos vectoriales

### 🔮 [agno](https://github.com/tu-usuario/lab-genai/tree/agno) - Agno
**Framework para agentes y multiagentes autónomos**
- Agentes autónomos e independientes
- Planificación y ejecución de tareas
- Razonamiento y toma de decisiones
- Integración con APIs externas
- Sistemas de retroalimentación

## 🚀 Inicio Rápido

### Prerrequisitos
```bash
# Python 3.8 o superior
python --version

# pip actualizado
pip install --upgrade pip
```

### Instalación
```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/lab-genai.git
cd lab-genai

# Instalar dependencias básicas
pip install -r requirements.txt
```

### Navegación entre ramas
```bash
# Ver todas las ramas disponibles
git branch -a

# Cambiar a una rama específica
git checkout autogen    # Para AutoGen
git checkout crewai     # Para CrewAI
git checkout langGraph  # Para LangGraph
git checkout agno       # Para Agno
git checkout main       # Para volver a Python básico
```
